Operadores Relacionais:

Servem para comparar dois valores, que podem ser variaveis
 ou constantes (desde que sejam do mesmo  valor):

 = (igual a);
 > (maior que);
 < (menor que);
 <> (diferente de);
 >= (maior ou igual a );
 <= (menor ou igual a);

Serão muito ultilizador quando o resultado destas 
comparações implicam em um valor logico.
------------------------------------------------------------------
Operadores Logicos:

os principais operadoes logicos são:

E - Conjunção (operador binário);
OU - Disjunção (operador binario);
Nâo - Negação (Operador unário);

Operadores logicos podem ser combinados de inúmeras formas, 
mas como os operadoes matematicos
eles seguem a uma ordem de precendÊncia:

primeiro:   () - parenteses;
segundo:    Não - negação;
terceiro:   E - Conjunção;
Quarto:     OU - Disjunção

Estruturas de Controle:

Existem três estruturas de controle;
 - sequencia simples;
 - Seleção;
 - repetição;


Sequencia simples: 
    é uma tarefa que sempre será execultada ma prde, da esquerda 
    para a direita e de cima para baixo:

Seleção:
    É ultilizado quando ha necessidade de escolhar entre opções.
    esta estrutura acarretará na execulção ou não de alguma tarefa,
    
	As formas ultilizadas para estas escolhar são:

	Condicionais:
	"Se... Então"
	"Se... Então..Senão..."
	

Repetição: 
    server para execultar uma rotina unúmeras vezes. essa repetição
pode ser finita ou infinita (tomar muito cuidado ao criar repetições
infinitas).

        Repetição:
	"Enquanto... faça"
	(no mundo da programação esta estrutura e conhecida pelo comando "white.. do..")

Eventos em JavaScript ---------------------------------------------------

São ocorrencias durante a execução do sistema a partir das quais 
o programador define as ações a serem realizadas pelo programa.

Explicaçoes sobre os eventos:

	onload - Ocorre no carregamento do documento, ou seja, 
	ocorre apenas no BODY do documento;

	Onuload - Ocorre na descarga/saida do documento. também ocorre penas no BOdy;

	onchange - ocorre quando o objeto perde o focus e houve mudança no conteúdo
	(válido para text, select e textarea.)

	onblur - ocorre quando o objeto perde o focus, indepedente de ter havido mudança ou não 
	(válido para text, select e textárea);

	onfocus - ocorre quando o objeto recebe o focus (válido para text, select, textárea)

	onclick - Ocorre quando o objeto recebe um click do mouse 
	(válido para buttom, checkbox, radio, link, reset e submit);

	onmouseover - ocorre quando o ponteiro do mouse passa sobre o objeto (válido apenas para link);

	onselect - ocorre quando o objeto é selecionado (válido para text e textarea);

	onsubmit - ocorre quando um botão do tipo submit recebe um click do moude (válido apenas em form.)

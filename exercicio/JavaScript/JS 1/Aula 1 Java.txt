-=------- ABRINDO NOVAS JANELAS----------------------------------

	A sintaxe padrão para este método é:

	variavel = window.open("url", "Nome da janela", "Opções")
	onde:

	Variavel - nome que será atribuido como propriedade da janela;
	url - endereço internet a ser aberto na janela;
	nome da janela - É o nome que aparecerá mp topo "Título";
	Opções - Definem as caracteristicas dessa janela, elas
	São: 
		toolbar - cria uma barra de ferramentas "back", "forward", etc...
		Location - abre a barra location do navegador;
		status - barra de status no rodapé da janela;
		scrollbars - Barras de rolagem;
		menubar - cria uma barra de menu;
		resizable - possibilita o redimensionamento;
		width - define a largura da janela (pixels)
		heigt - define a altura da janela (pixels)
		//Exeto as opções width e height que aceitam parãmetros em pixels, as
		demais são booleanas (1 para verdadeiro e 0 para falso, exemplo:)

-------------ABRINDO NOVAS JANELAS -----------------------

	PARA abrir um documento dentro da janela, utilizamos a seguinte método e padrão:
	variavel.document.open()
	
	ESCREVENDO NO DOCUMENTO
	
	Para escrever a tela no documento, ultilizamos o seguinte método e padrão:
	variavel.document.write("Comando HTML, comandos JavaScript, textos, etc.")
	
	FECHANDO A JANELA
	
	Para fechar o janela utilizamos o seguinte método:
	variavel.document.close()
